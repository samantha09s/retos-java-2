<a name="ejecutar"></a>

## 📄 Ejecutar los retos localmente

```bash
git clone https://github.com/samantha09s/retos-java-2.git
cd retos-java-2/reto-01

# Compila los archivos
javac *.java

# Ejecuta el código
java Main
```

> ☕ Requiere Java 17 o superior.
